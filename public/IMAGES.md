# Place Your Images Here

Add the following image files to this directory:

- `hero-bakery.jpg` - Hero section background image (recommended: 1920x1080 or larger)
- `product-bread.jpg` - Artisan Breads section (recommended: 800x600)
- `product-pastries.jpg` - French Pastries section (recommended: 800x600)
- `product-cakes.jpg` - Celebration Cakes section (recommended: 800x600)
- `product-cookies.jpg` - Cookies & Biscotti section (recommended: 800x600)

## Image Recommendations

- **Hero Image**: Use a high-quality bakery image with good lighting
- **Product Images**: Show your baked goods in the best light - studio photography recommended
- **Format**: JPEG or PNG (JPEG for photos, PNG for graphics)
- **Optimization**: Compress images to reduce page load time (use tools like TinyPNG)

The HTML is already configured to use these images from this `public/` directory.
